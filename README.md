# Fake News Classification Project

This project focuses on categorising news articles as either real or fake using Natural Language Processing (NLP) techniques. The analysis and modelling are conducted within a Jupyter Notebook environment.

## Project Overview

The goal of this project is to build a classification model that can accurately distinguish between legitimate news and misinformation. The workflow involves cleaning raw text data, performing exploratory data analysis, and training a machine learning model to recognise patterns indicative of "fake news."

## Tech Stack

* **Language:** Python
* **Analysis:** Pandas
* **Visualisation:** Matplotlib, Seaborn
* **Preprocessing:** `re` (Regular Expressions), NLTK, Gensim
* **Machine Learning:** Scikit-learn (Sklearn)
* **Environment:** Jupyter Notebook
